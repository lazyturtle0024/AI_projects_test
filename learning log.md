# GitHub Learning Log

## What I learned today

- Git is version control.
- GitHub is a cloud platform for Git projects.
- A repo is a project folder with history.
- A commit is a saved version checkpoint with a message.
- GitHub Pages can publish an HTML file as a website.

## Key insight

GitHub is not just file storage.
It is a project memory system.
